# Santa Slay 3D (AI‑First)

A web‑first, high‑fidelity 3D downhill runner for ages 7+, designed to be easily packaged for iOS.

## Quick start (Web)

### Prereqs
- Node.js 18+ recommended

### Run
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
npm run preview
```

## Repo truth
- **SPEC.md** — canonical game spec
- **AGENTS.md** — rules for GPT‑5.2 and other AI agents
- **PLAN.md** — milestone plan + task checklist
- **MEMORY.md** — compact project memory (kept short; updated frequently)
- **DECISIONS.md** — record of architectural decisions

## Safety note
This project targets kids. Do not add tracking, ads, chat, outbound links, or UGC without explicit human approval and a compliance review.
